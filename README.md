# SQL Portfolio

A collection of SQL projects covering business analytics, data reporting, and query optimization.
Built using PostgreSQL. Topics include aggregations, joins, window functions, CTR analysis, and engagement metrics.

---

## Projects

### 1. Average Selling Price Analysis
**File:** `1_AVERAGE_SELLING_PRICE.sql`  
**Concepts:** GROUP BY, AVG, aggregate functions  
**Summary:** Calculates average selling price per product from a multi-entry sales table.

---

### 2. Bank Account Summary
**File:** `2_BANK_ACCOUNT_SUMMAY.sql`  
**Concepts:** CASE WHEN, SUM, GROUP BY  
**Summary:** Aggregates deposits and withdrawals per account holder to calculate net balance.

---

### 3. Customers Without Orders
**File:** `3_CUSTOMERS.sql`  
**Concepts:** LEFT OUTER JOIN, NULL filtering  
**Summary:** Identifies customers who have never placed an order using a LEFT JOIN with NULL check.

---

### 4. Market Analysis
**File:** `4_MARKET_ANLYSIS.sql`  
**Concepts:** GROUP BY, ORDER BY, SUM, revenue aggregation  
**Summary:** Analyzes total revenue and quantity sold by product and customer; ranks top customers by spend.

---

### 5. Product Sales Analysis
**File:** `5_PRODUCT_SALES_ANALYSIS.sql`  
**Concepts:** INNER JOIN, SUM, GROUP BY  
**Summary:** Joins product and orders tables to calculate total revenue per product.

---

### 7. Email Duplicate Detection
**File:** `7_EMAIL-DUPLICATES.sql`  
**Concepts:** DISTINCT ON, deduplication  
**Summary:** Detects and removes duplicate email entries, retaining the earliest record per email address.

---

### 8. Name Formatting
**File:** `8_NAMES.sql`  
**Concepts:** INITCAP, string functions  
**Summary:** Standardizes inconsistently cased name data into proper title case using INITCAP.

---

## Tools Used
- PostgreSQL
- SQL (DDL + DML + DQL)
- Joins, Aggregations, Window Concepts, String Functions, Date Functions
